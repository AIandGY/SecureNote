# SecureNote：安全 · 智能 · 四象限效率工具

SecureNote 是一款专为 Android 打造的隐私优先效率应用，集成了**四象限笔记**、**生物识别加密密码本**与**智能 AI 语音助手**。它基于艾森豪威尔矩阵管理任务，结合百度语音识别与 DeepSeek 大模型，提供硬件级安全防护。
---

## 📌 功能特性

### 📓 智能四象限笔记本 (Material Design 3)
*   **动态优先级管理**：系统根据“重要”与“紧急”维度自动分类，并在主页以四象限逻辑展示：
    *   🔴 **重要且紧急**：红色标识，最高优先级。
    *   🔵 **重要不紧急**：蓝色标识。
    *   🟡 **紧急不重要**：黄色标识。
    *   ⚪ **不重要不紧急**：灰色标识。
*   **智能演变逻辑**：任务进入“提前提醒”窗口期或到达设定时间时，系统会自动计算**有效优先级 (Effective Importance)**，动态提升显示权重。
*   **多维交互**：
    *   支持列表与瀑布流切换。
    *   **原生排序与筛选**：通过工具栏图标实现快速切换（支持按时间、标题、优先级排序）。
    *   **批量操作**：长按进入选择模式，支持多选删除或状态更改。

### 🤖 AI 智能语音录入
*   **百度语音驱动**：集成百度语音识别系统，提供高精度语音转文字。
*   **DeepSeek 语义处理**：**长按**右下角编写按钮触发 AI 模式。自动提取：标题、内容、优先级、提醒日期及具体时间点，实现“一句话创建任务”。
*   **实时反馈**：具备震动反馈、呼吸灯动画（Pulse Animation）以及 Snackbar 状态提示。

### 🔒 安全密码本
*   **生物识别防护**：调用系统级 `BiometricPrompt`（指纹/人脸），确保私密数据仅本人可访问。
*   **安全认证流**：认证取消或失败时自动退回主页面，保护内存数据。
*   **快速交互**：支持密码一键复制、明文/密文快速切换。

### 📅 日程与情境记录
*   **周视图日程表**：以周为单位展示待办，支持“上周/下周”动态标题切换。
*   **智能环境戳**：
    *   **地理位置**：基于原生 `Geocoder` 自动获取当前位置（如：成都市·武侯区）。
    *   **实时天气**：对接和风天气 API，随笔记保存当时的天气与气温。
    *   **时间戳**：精准记录创建时刻。

## 🛠 技术栈

| 模块 | 技术实现 |
| :--- | :--- |
| **核心框架** | Java (Android Native) / AppCompat 1.6.1 |
| **UI 组件** | Material Design 3 / ViewBinding / ConstraintLayout |

| **定位/天气** | Android Location Services + Geocoder + 和风天气 API |

## 🚀 快速开始

### 环境要求
*   Android 10.0 (API 29) 或更高版本。
*   设备支持指纹识别或面部识别。

### 权限需求
*   `RECORD_AUDIO`：用于 AI 语音输入。
*   `ACCESS_FINE_LOCATION`：用于获取位置戳与天气。
*   `VIBRATE`：用于长按反馈。
*   `POST_NOTIFICATIONS`：用于任务提醒推送。

### API 配置
1.  **AI 设置**：在“设置 -> API接口设置 -> AI助手”中填入 DeepSeek API Key。
2.  **天气设置**：在“设置”中开启显示天气，并在“设置 -> API接口设置 -> 和风天气”中填入和风天气 Key。
3.  **语音识别**：在“设置 -> API接口设置 -> 百度语音识别”中填入百度语音识别 `AppID`、`API Key` 已正确配置。

## 📖 使用指南

### 语音创建笔记
1.  在主页面**长按**右下角的“写笔记”浮动按钮。
2.  触发震动反馈后开始说话，例如：“明天下午三点提醒我开会，这件事非常重要”。
3.  松开按钮，AI 将自动分析并创建带提醒、带优先级、带环境戳的任务。

### 排序与高级筛选 (核心操作)
*   **单击图标**：弹出对话框，选择笔记的**排序方式**（时间、标题、优先级）。
*   **长按图标**：触发**高级优先级筛选**，可选择只显示“重要且紧急”等特定象限的任务。

### 密码本管理
*   点击底部导航栏进入“密码本”标签。
*   系统将强制调起指纹/面部验证，验证通过后方可显示内容。

## 🤝 贡献与维护
*   遵循 **Google Java Style** 代码规范。
*   **ViewBinding**：所有布局引用需通过 `binding` 对象，严禁使用 `findViewById`。
*   **数据库**：修改 `Note` 实体类时请务必更新 Room 数据库版本号。
