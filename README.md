# SecureNote：安全 · 智能 · 四象限效率工具

SecureNote 是一款专为 Android 平台打造的隐私优先型效率应用。它深度集成了四象限笔记逻辑、生物识别加密密码本与智能 AI 语音助手。通过艾森豪威尔矩阵管理任务，结合百度语音识别与 DeepSeek 大模型处理，为用户提供硬件级的安全防护与智能化记录体验。

## 📌 功能特性

### 📓 智能四象限笔记本 (Material Design 3)

- **动态优先级管理**：系统根据“重要”与“紧急”维度自动分类，并在主页以四象限逻辑展示：
  - 🔴 **重要且紧急**：红色标识，最高优先级处理。
  - 🔵 **重要不紧急**：蓝色标识，侧重长期规划。
  - 🟡 **紧急不重要**：黄色标识，关注即时处理。
  - ⚪ **不重要不紧急**：灰色标识，静默管理。
- **有效优先级 (Effective Importance)**：系统会根据任务的“提前提醒时间”与“开始时间”自动计算。当任务临近时，权重会自动动态提升。
- **原生交互体验**：
  - 拖拽动画：采用原生 ItemTouchHelper 实现，拖拽过程中伴随半透明视觉反馈。
  - 多维筛选：支持按时间、标题、优先级排序，并可长按筛选图标进行单象限过滤。

### 🤖 AI 智能语音录入

- **百度语音驱动**：集成百度语音 SDK，支持高精度的实时语音转文字。
- **一句话创建任务**：长按编写按钮触发 AI 模式，调用 DeepSeek 大模型自动提取语音中的：标题、正文、重要程度、提醒日期及时间。
- **触觉反馈**：具备线性震动反馈与呼吸灯动画（Pulse Animation），确保人机交互的实时性。

### 🔒 生物识别密码本

- **硬件级安全**：调用系统 BiometricPrompt 接口（指纹/人脸），确保私密数据仅在生物认证通过后从内存解密。
- **安全认证流**：认证取消或失败时自动退回主页面，严防数据残留。
- **便捷管理**：支持密码一键复制、明文/密文切换，条目显示采用安全遮罩。

### 💾 数据管理与救援

- **完整导出/导入**：支持将笔记与密码本数据打包导出为本地加密备份文件，方便跨设备迁移或数据备份。
- **数据库救援模式 (Rescue Mode)**：
  - 冲突自动捕获：当系统检测到数据库版本冲突或结构损坏时，会自动拦截闪退并引导进入救援界面。
  - 残留数据导出：支持在数据库重建前，手动尝试导出残留的文本数据，将损失降到最低。
  - 自动索引修复：提供数据库重建与索引清理功能，恢复应用可用性。

## 🚀 快速开始

### 环境要求

- Android 10.0 (API 29) 或更高版本。
- 设备需支持指纹识别、人脸识别或锁屏密码。

### 权限配置

1. `RECORD_AUDIO`：用于 AI 语音输入。
2. `ACCESS_FINE_LOCATION`：用于自动补全位置戳与天气。
3. `VIBRATE`：用于长按触发与报警反馈。
4. `POST_NOTIFICATIONS`：用于高优先级任务提醒。

### API 接口设置

- 进入“设置 -> API 接口设置”：
  - AI 助手：填入 DeepSeek API Key。
  - 天气服务：填入和风天气 API Key。
  - 语音识别：内置百度语音识别，请确保 AppID/API Key 已在清单文件中正确配置。

## 📖 使用指南

### 语音创建任务

1. 在主页面长按右下角的浮动按钮（FloatingActionButton）。
2. 震动后开始说话（例如：“明天早上九点提醒我开会，这非常紧急”）。
3. 松开按钮，AI 将自动分析情境并创建任务，同时补全当前的时间、地点、天气标签。

### 数据备份与恢复

1. 前往“设置 -> 数据管理”。
2. 点击“导出数据”生成备份文件。
3. 在新设备点击“导入数据”并选择该文件即可恢复。
4. 导出的加密密码本可以在新的设备上输入恢复密钥进行恢复数据

### 数据库救援

- 若应用因数据库更新出现异常，系统会提示“检测到数据库冲突”。
- 进入救援模式，点击“尝试导出文本”备份现有内容，然后选择“重置数据库”修复结构。
